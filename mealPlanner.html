Project Walkthrough - Meal Planner App

Personal Project – Progressive Web App with React

← Back to Main Screen

This Meal Planner App is a mobile-friendly progressive web app designed to help users plan meals, manage pantry inventory, filter recipes by allergies, and generate dynamic shopping lists that scale with portion sizes. It supports saving custom recipes, works offline, and includes a dark mode for comfortable viewing.

Tech Stack

  • Languages: JavaScript, HTML, CSS
  • Framework: React
  • Build Tool: Vite
  • Data Storage: localStorage
  • Environment: Localhost (runs on development server)

Project Functionality

This section highlights the main features of the Meal Planner App, including the weekly planner, recipe filtering, shopping list generation, and pantry inventory management.

Weekly Meal Planner

Plan breakfasts, lunches, dinners, and snacks for the week in a clear, mobile-friendly layout.

Recipe Filtering & Search

Find recipes based on available ingredients, allergy preferences, and tags like 'healthy' or 'quick'.

Smart Shopping List

Automatically generate a shopping list from planned meals, with quantities scaled to the number of portions.

Pantry Inventory

Track what ingredients you already have to reduce waste and save money.

Offline & Installable

As a PWA, the app can be installed on devices and used offline for convenience.
